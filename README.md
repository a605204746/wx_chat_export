# 企业微信消息采集工具 · 聊天记录导出 Excel

> 无需开启会话存档 · 支持群聊/私聊 · 批量采集 · 一键导出 Excel

企微聊天记录导出工具，通过 RPA 模拟操作自动采集企业微信群消息与私聊消息，并导出为 Excel 文件。支持批量多对象采集、截止日期增量抓取、全程无人值守，适用于企业合规存档、客服质检、数据分析等场景。

## 功能特性

- **实时消息采集** — 自动定位企业微信聊天窗口，滚动翻页提取群聊/私聊消息，实时写入本地数据库
- **批量无人值守** — 上传 Excel 目标列表，按序自动切换群/联系人逐一采集，全程无需人工干预
- **聊天记录导出 Excel** — 导出带表头、交替行色、边框、自适应列宽的规范报表，开箱即用
- **可视化区域框选** — 全屏暗色遮罩 + 鼠标拖拽，精确框定聊天消息区域
- **截止日期过滤** — 仅采集指定日期之后的消息，增量抓取、零重复
- **配置自动持久化** — 坐标、参数、截止日期等所有配置自动保存，重启即恢复
- **企微运行状态监测** — 实时轮询企微进程，运行状态一目了然

## 快速开始

### 系统要求

- Windows 10 / 11（64 位）
- 企业微信（WXWork）已安装并登录
- 采集期间企业微信窗口需保持**可见状态**（不可最小化至系统托盘）

### 安装与启动

```bash
git clone https://gitee.com/shendudian/wx_chat_export.git
```

克隆完成后，双击目录中的 `wx_rpa_chat-win-amd64.exe` 即可启动，无需安装任何依赖。

### 采集流程

1. 确认右上角状态灯为**绿色**（企微运行中）
2. 输入采集对象名称（群名或联系人，须与企微侧边栏**完全一致**），或切换至「批量采集」上传目标列表
3. 点击「鼠标框选聊天区域」，屏幕变暗后拖拽框住消息区域（**不含输入框**）
4. 按需调整采集参数：翻页次数、滚动量、翻页间隔、停止阈值
5. 点击「开始采集」，工具自动翻页抓取并实时显示进度
6. 采集完成后在「消息列表」查看数据，点击「导出 Excel」保存

> **快捷键**：采集过程中按 `F8` 可立即停止。

### 批量采集

1. 点击「下载模板」获取 Excel 模板
2. 在「采集对象」列填入所有群名或联系人名称
3. 点击「上传列表」导入
4. 点击「开始采集」，工具按顺序逐一切换并采集

## 采集参数说明

| 参数 | 含义 | 默认值 | 范围 |
|------|------|--------|------|
| 每批翻页次数 | 每轮采集中向上滚动的次数 | 3 | 1–50 |
| 单次滚动量 | 每次滚动的幅度 | 10 | 1–200 |
| 翻页间隔 | 两次滚动之间的等待时间 | 1.5 秒 | 0.1–5.0 |
| 停止阈值 | 连续多少批内容未变化时自动停止 | 4 | 1–20 |
| 截止日期 | 仅采集此日期之后的消息（可选） | 不限 | YYYY-MM-DD |

> 消息量大时可增大翻页次数和滚动量；企微响应较慢时适当增加翻页间隔。

## 常见问题

| 问题 | 解决方式 |
|------|----------|
| 采集无内容 | 检查框选区域是否覆盖了消息列表（不含输入框） |
| 提示「未找到企业微信」 | 确保企微窗口未最小化至系统托盘，保持可见状态 |
| 导航失败跳过 | 群名须与企微侧边栏名称完全一致，注意空格与特殊字符 |
| 坐标偏移 | 高 DPI 屏幕已自动适配；调整企微窗口大小后需重新框选 |

> 遇到其他问题，可在「系统日志」查看详细错误信息后反馈给开发者。
